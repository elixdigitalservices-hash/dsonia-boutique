## 4. Paleta de colores recomendada

La identidad visual actual tira mucho de **negro + blanco** por el logo, y las fotos meten códigos visuales de **burdeos, rojo, crema, dorado, terracota y tonos andaluces**.

Como la web no expone una guía de marca formal, esta paleta está inferida de logo, fotos y estética actual.

| Uso | Color | HEX recomendado |
|---|---:|---|
| Negro principal / lujo | Negro boutique | `#050505` |
| Blanco logo / textos sobre oscuro | Blanco puro | `#FFFFFF` |
| Fondo elegante cálido | Crema suave | `#F7F1E7` |
| Fondo secundario | Beige moda | `#E8D9C5` |
| Acento principal | Burdeos elegante | `#7A1021` |
| Acento flamenco | Rojo pasión | `#C8102E` |
| Acento andaluz | Terracota | `#B85C45` |
| Detalles premium | Dorado champán | `#C5A46D` |
| Texto secundario | Gris cálido | `#68625D` |
| Verde/petróleo de contraste | Verde noche | `#1F3430` |

**Recomendación estratégica:**  
No hacer una web rosa típica de boutique. Mejor usar **negro boutique + crema + burdeos + dorado sutil**. Eso sube percepción premium sin perder cercanía.

---

## 5. Tipografías recomendadas

Para que la web parezca una boutique de verdad y no una plantilla sin terminar:

| Uso | Fuente recomendada |
|---|---|
| Titulares elegantes | `Playfair Display`, `Cormorant Garamond` o `Bodoni Moda` |
| Texto normal | `Montserrat`, `Lato` o `Inter` |
| Detalles/acento | Fuente script elegante solo para pequeños detalles, nunca para párrafos largos |

## 9. Recomendación final

No conviene copiar la web actual tal cual. Hay que usarla como materia prima.

La nueva web debería estar mucho más enfocada a:

- Conversión por WhatsApp.
- SEO local.
- Galería visual.
- Confianza.
- Visitas a tienda física.
- Moda flamenca y eventos como secciones potentes.
- Posibilidad futura de catálogo online o ecommerce.

La estrategia clara es que una clienta vea un look, conecte emocionalmente y pulse WhatsApp o vaya directamente a tienda. Ahí está la caja.
